Documentación de HTML y CSS

Introducción

En esta documentación, aprenderás los conceptos básicos de HTML (HyperText Markup Language) y CSS (Cascading Style Sheets), dos tecnologías fundamentales para el desarrollo web. HTML se utiliza para estructurar el contenido de una página web, mientras que CSS se encarga de su presentación.

HTML Básico

HTML es el lenguaje de marcado que se utiliza para crear la estructura de una página web. Aquí, aprenderás sobre etiquetas, elementos y cómo crear una estructura básica de página web. Aquí tienes un ejemplo básico que muestra como va estructurado el HTML: 

                
                    

                        <!DOCTYPE html>
                        <html>
                            <head>
                                <title>Mi Página</title>
                            </head>

                            <body>
                                <h1>Bienvenido</h1>
                                <p>Esta es una página web de ejemplo.</p>
                            </body>
                        </html>
Etiquetas y Elementos HTML

Las etiquetas y elementos HTML son componentes fundamentales para crear páginas web. A continuacion, vamos a aprender sus conceptos y veremos ejemplos de como usarlos.

Etiqueta HTML

Una etiqueta HTML es una instrucción que marca un elemento en un documento HTML. Las etiquetas están compuestas por corchetes angulares < elemento >. Las etiquetas se utilizan para definir la estructura y el contenido de una página web.

Elemento HTML

Un elemento HTML es un conjunto completo que incluye la etiqueta de apertura, el contenido y la etiqueta de cierre. Los elementos HTML se utilizan para representar diferentes tipos de contenido, como texto, imágenes, enlaces y más. Ejemplo de elemento HTML: 

                    
                        

                            <p>Este es un párrafo de ejemplo.</p>

Atributo HTML

Los atributos son información adicional proporcionada dentro de una etiqueta para configurar o modificar el comportamiento de un elemento. Los atributos se colocan dentro de la etiqueta de apertura y suelen tener un nombre y un valor. Ejemplo de atributo: 

                    
                        

                            <img src="imagen.jpg" alt="Descripción de la imagen">
                       
                            En este ejemplo "src" y "alt" son los atributos

Contenido HTML

El contenido de un elemento HTML es la información que se muestra en la página. Puede ser texto, imágenes, enlaces o incluso otros elementos HTML anidados. Ejemplo de contenido HTML: 

                    
                        

                            Texto:
                            <p>Soy el contenido de esta etiqueta.</p>   
                            
                            Enlace que contiene imagen anidada:
                            <a>
                                <img src="URL_de_la_imagen" alt="Ejemplo">
                            </a>

Estructura HTML

La estructura HTML se refiere a la forma en que se organizan y anidan las etiquetas y elementos en un documento HTML. La estructura adecuada es importante para que el navegador interprete correctamente la página web.

Página HTML

Una página HTML es un documento que contiene código HTML y se muestra en un navegador web. Es la unidad básica de contenido en la web.

Comentario

Un comentario es una porción de texto dentro de un documento o código que no se muestra en la página web final. Los comentarios se utilizan para incluir notas, explicaciones o aclaraciones para los desarrolladores que están trabajando en el código o para proporcionar información adicional en la documentación. 

                    
                        

                            <!--Esto es un comentario-->

Etiquetas y Elementos HTML Básicos

  • `<html>`: Define el documento HTML.
  • `<head>`: Contiene información sobre el documento, como el título de la página.
  • `<body>`: Contiene el contenido visible de la página.
  • `<h1>`: Define un encabezado de nivel 1.
  • `<p>`: Define un párrafo de texto.
  • `<a>`: Crea un hipervínculo.
  • `<img>`: Inserta una imagen.

A continuación, una demostración sencilla de cómo se ven y se usan estos elementos en HTML:

                    
                        

                            <!DOCTYPE html>
                            <html>
                                <head>
                                    <title>Mi Primera Página</title>
                                </head>
                            
                                <body>
                                    <h1>Este es un Título de Nivel 1</h1>
                                    <p>Este es un párrafo. Puedes escribir cualquier contenido aquí.</p>
                                    <a href="https://www.ejemplo.com">Enlace a Ejemplo.com</a>
                                    <img src="imagen.jpg" alt="Descripción de la imagen">
                                </body>
                            </html>

En este ejemplo:

  • `<html>` envuelve todo el contenido de la página.
  • `<head>` contiene metainformación, como el título de la página.
  • `<body>` contiene el contenido visible, que incluye un título `<h1>`, un párrafo `<p>`, un enlace `<a>`, y una imagen `<img>`.
  • El enlace `<a>` lleva a "https://www.ejemplo.com".
  • La etiqueta de imagen `<img>` muestra la imagen "imagen.jpg" con una descripción alternativa.
HTML ofrece muchas más etiquetas y elementos para crear páginas web más completas y ricas en contenido.
Formularios HTML

Los formularios HTML son elementos que permiten a los usuarios ingresar datos y enviar esa información a un servidor web para su procesamiento. Los formularios son una parte fundamental de la interacción en la web y se utilizan para recopilar datos como nombres de usuario, contraseñas, comentarios, opciones de selección y mucho más. Aquí tienes un ejemplo básico de un formulario HTML: 

                
                    

                        <form action="URL_a_la_que_se_enviarán_los_datos" method="post">
                            <label for="nombre>Nombre:</label>
                            <input type="text" id="nombre" name="nombre" required>
                    
                            <label for="email>Correo electrónico:</label>
                            <input type="email" id="email" name="email" required>
                    
                            <label for="mensaje>Mensaje:</label>
                            <textarea id="mensaje" name="mensaje" rows="4" required></textarea>
                    
                            <input type="submit" value="Enviar>
                        </form>

En este ejemplo:

  • <form>: Se define el elemento <form> que contiene todos los elementos del formulario. El atributo action especifica la URL a la que se enviarán los datos del formulario y el atributo method determina el método de envío (en este caso, "post").
  • <label>: Los elementos <label> se utilizan para etiquetar los campos del formulario y mejorar la accesibilidad. El atributo for del <label> se relaciona con el atributo id del campo correspondiente.
  • <input>: Los elementos <input> se utilizan para recopilar información como texto o direcciones de correo electrónico. En este ejemplo, hay un campo de texto y un campo de correo electrónico.
  • <textarea>: El elemento <textarea> se utiliza para recopilar texto largo, como comentarios o mensajes extensos.
  • <input type="submit">: Este botón de tipo "submit" se utiliza para enviar el formulario al servidor. Cuando el usuario hace clic en este botón, los datos del formulario se envían a la URL especificada en el atributo action.
El ejemplo anterior es un formulario básico, pero los formularios HTML pueden incluir una amplia variedad de elementos y atributos para satisfacer diferentes necesidades. Los datos recopilados en el formulario pueden procesarse en el lado del servidor utilizando lenguajes como PHP, Python o JavaScript, entre otros.
CSS Básico

CSS, o Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje de diseño utilizado para controlar la presentación y el aspecto visual de documentos HTML. En otras palabras, CSS se utiliza para definir cómo se verá una página web, incluyendo la tipografía, colores, márgenes, tamaños de texto, posicionamiento de elementos y más. Veamos un ejemplo muy básico de un archivo HTML con un estilo CSS incorporado en la misma página: 

                
                    

                        <html>
                            <head>
                                <title>Ejemplo CSS</title>

                                <style>
                                    p {
                                        color: blue;
                                        font-size: 16px;
                                    }
                                    .destacado {
                                        background-color: yellow;
                                        font-weight: bold;
                                    }
                                </style>
                            </head>
                        
                            <body>
                                <p>Este es un párrafo normal.</p>
                    
                                <p class="destacado">Este es un párrafo destacado.</p>
                            </body>
                        </html>

En este ejemplo:

  • El estilo se define dentro de la sección <style> en la cabecera del documento HTML.
  • Se aplican estilos a todos los elementos <p> para cambiar el color del texto y el tamaño de la fuente.
  • Se aplica un estilo adicional al párrafo con la clase "destacado" para cambiar el fondo y hacerlo negrita.
CSS es un lenguaje poderoso que permite un control detallado sobre la presentación de una página web. Los estilos también pueden definirse en archivos externos para facilitar la reutilización y el mantenimiento del diseño.
Selectores CSS

Los selectores CSS son patrones que se utilizan para seleccionar y aplicar estilos a elementos específicos en un documento HTML. Los selectores son fundamentales en CSS, ya que determinan qué elementos en la página web se verán afectados por las reglas de estilo que definas. Aquí hay algunos selectores CSS comunes:

Selector de tipo

Selecciona elementos basados en su etiqueta HTML. Por ejemplo, p seleccionará todos los párrafos en la página. 

                    
                        

                            p {
                                color: blue;
                            }

Selector de clase

Selecciona elementos basados en su atributo class. Los elementos con la clase especificada se verán afectados. 

                    
                        

                            .destacado {
                                background-color: yellow;
                            }

Selector de ID

Selecciona un elemento único basado en su atributo id. Los IDs deben ser únicos en la página. 

                    
                        

                            #encabezado {
                                font-size: 24px;
                            }

Selector de descendencia

Selecciona elementos que son descendientes de otro elemento. Por ejemplo, ul li seleccionará todos los elementos <li> que están dentro de una lista desordenada (<ul>). 

                    
                        

                            ul li {
                                list-style-type: square;
                            }

Selector de grupo

Permite agrupar varios selectores y aplicar el mismo estilo a todos ellos. 

                    
                        

                            h1, h2  {
                                color: red;
                            }

Selector de atributo

Selecciona elementos con un atributo HTML específico. Por ejemplo, [type="text"] seleccionará todos los elementos de entrada de texto. 

                    
                        

                            input[type="text"] {
                                border: 1px solid #ccc;
                            }

Selector de pseudo-clase

Selecciona elementos en función de su estado o posición. Por ejemplo, :hover selecciona un elemento cuando el cursor del mouse se encuentra sobre él. 

                    
                        

                            a:hover {
                                text-decoration: underline;
                            }

Selector de pseudo-elemento

Selecciona una parte específica de un elemento. Por ejemplo, ::before crea un elemento antes del contenido de un elemento seleccionado. 

                    
                        

                            p::before {
                                content: "Antes de: ";
                            }

Estos son solo algunos ejemplos de selectores CSS. Los selectores te permiten apuntar de manera precisa a elementos en tu página web y aplicar estilos de forma selectiva. Puedes combinar selectores para lograr reglas de estilo más complejas y personalizadas.

Diseño Responsivo

El diseño responsivo (también conocido como diseño adaptable o responsive design en inglés) es una técnica de diseño web que permite que un sitio web se adapte de manera automática a diferentes tamaños de pantalla y dispositivos, como computadoras de escritorio, tabletas y teléfonos móviles. Esto garantiza que la experiencia del usuario sea óptima sin importar el dispositivo que esté utilizando. CSS desempeña un papel fundamental en la implementación del diseño responsivo. Ejemplos sencillos:

Media Queries

Las media queries son reglas de CSS que permiten aplicar estilos específicos según las características de la pantalla del dispositivo. Pueden depender de factores como el ancho de la pantalla, la altura de la pantalla, la orientación del dispositivo, la resolución, etc. Por ejemplo, para cambiar la fuente y el tamaño de texto en pantallas más pequeñas, puedes hacer lo siguiente: 

                    
                        

                            @media (max-width: 768px) {
                                body {
                                    font-size: 16px;
                                }
                            }
    
                            
                En este ejemplo, cuando el ancho de la pantalla sea de 768 píxeles o menos, 
                el tamaño de fuente del cuerpo se reducirá a 16 píxeles.

Viewport

En el diseño responsivo, se utiliza el valor viewport para establecer la escala inicial de la página. Esto permite que la página se ajuste correctamente en la pantalla del dispositivo. Puedes usar la siguiente línea de código en la sección <head> de tu documento HTML: 

                    
                        

                            <meta name="viewport" content="width=device-width, initial-scale=1.0">

Fluid Grids

En lugar de especificar tamaños fijos para elementos en tu diseño, como anchos de caja en píxeles, se utilizan porcentajes para crear diseños fluidos que se adaptan al tamaño de la pantalla. Por ejemplo, puedes definir un contenedor principal con un ancho del 100%: 

                    
                        

                            .container {
                                width: 100%;
                            }

Imágenes flexibles

Las imágenes pueden redimensionarse de manera fluida utilizando CSS para asegurarse de que no se desborden en pantallas más pequeñas. Esto se logra mediante la propiedad max-width: 100%;: 

                    
                        

                            img {
                                max-width: 100%;
                                height: auto;
                            }
Modelos de Caja en CSS

Los "modelos de caja" en CSS se refieren a cómo se representa y se comporta un elemento HTML en una página web en términos de su contenido, relleno, bordes y margen. En CSS, cada elemento en una página se considera una "caja" que consta de cuatro componentes principales: contenido, relleno, borde y margen. Estos componentes se pueden controlar y modificar mediante propiedades de CSS.

Contenido (Content)

El contenido es la parte real y visible del elemento, como el texto o las imágenes. Se controla con la propiedad `width` para establecer el ancho y `height` para establecer la altura del contenido. 

                    
                        

                            .caja {
                                width: 200px;
                                height: 100px;
                            }

Relleno (Padding)

El relleno es el espacio entre el contenido y el borde del elemento. Se controla con las propiedades `padding-top`, `padding-right`, `padding-bottom` y `padding-left`, o de manera abreviada con `padding`. 

                    
                        

                            .caja {
                                padding: 10px; /* 10px de relleno en todas las direcciones */
                            }

Borde (Border)

El borde rodea el contenido y el relleno del elemento. Se controla con las propiedades `border-width`, `border-style` y `border-color`. También se puede establecer de manera abreviada con `border`. 

                    
                        

                            .caja {
                                border: 2px solid #000; /* Borde de 2px de ancho, sólido y de color negro */
                            }

Margen (Margin)

El margen es el espacio entre el borde del elemento y otros elementos circundantes. Se controla con las propiedades `margin-top`, `margin-right`, `margin-bottom` y `margin-left`, o de manera abreviada con `margin`. 

                    
                        

                            .caja {
                                margin: 20px; /* 20px de margen en todas las direcciones */
                            }

Aquí tienes un ejemplo completo que muestra cómo se aplican estos componentes del modelo de caja en un elemento HTML: 

                
                    

                        <!DOCTYPE html>
                        <html>
                            <head>
                                <style>
                                    .caja {
                                        width: 200px;
                                        height: 100px;
                                        padding: 10px;
                                        border: 2px solid #000;
                                        margin: 20px;
                                    }
                                </style>
                            </head>
    
                            <body>
                                <div class="caja">
                                    Este es un ejemplo de modelo de caja en CSS.
                                </div>
                            </body>
                        </html>

En este ejemplo, la clase `.caja` se aplica a un `<div>` y se controlan sus dimensiones, relleno, borde y margen mediante CSS. Esto ilustra cómo puedes controlar el aspecto y el espacio alrededor de un elemento en tu página web utilizando el modelo de caja en CSS.

Tipografía en CSS

La tipografía en CSS se refiere a cómo puedes controlar y personalizar la apariencia del texto en tu sitio web utilizando hojas de estilo en cascada (CSS). Esto incluye la elección de la fuente, el tamaño, el espaciado entre letras y líneas, el peso de la fuente y otros aspectos relacionados con el aspecto del texto en tu página web. Ejemplos de cómo puedes aplicar estilos de tipografía en CSS:

Cambiar la fuente de un elemento

                    
                        

                            font-family: "Arial", sans-serif;

En este ejemplo, estamos estableciendo la fuente del texto en "Arial". Si "Arial" no está disponible en la computadora del usuario, se usará una fuente genérica "sans-serif" como respaldo.

Cambiar el tamaño de la fuente

                    
                        

                            font-size: 16px;

Aquí, estamos estableciendo el tamaño de la fuente en 16 píxeles.

Cambiar el grosor de la fuente

                    
                        

                            /* Cambiar el grosor de la fuente a negrita */
                            font-weight: bold;

Esto hará que el texto sea más grueso y, por lo tanto, "negrita".

Aplicar estilo cursiva a la fuente

                    
                        

                            font-style: italic;

Esto hará que el texto tenga un estilo cursiva.

Cambiar el color del texto

                    
                        

                            color: blue;

En este caso, el texto se mostrará en color azul.

Alinear el texto dentro de un contenedor

                    
                        

                            text-align: center;

Esto alineará el texto en el centro del contenedor.

Cambiar la altura de línea (interlineado)

                    
                        

                            line-height: 1.5;

Un valor de 1.5 significa que la altura de línea será un 150% del tamaño de la fuente actual.

Ajustar el espaciado entre caracteres (kerning)

                    
                        

                            letter-spacing: 2px;

En este caso, estamos aumentando el espaciado entre caracteres en 2 píxeles.

Estos son ejemplos básicos de cómo puedes controlar la tipografía en CSS. Puedes aplicar estas propiedades a elementos específicos en tu HTML utilizando selectores CSS para personalizar la apariencia de tu texto en tu sitio web.

Colores y Fondos en CSS

En CSS, puedes controlar los colores y fondos de elementos HTML para personalizar la apariencia de tu sitio web. Hay varias propiedades que te permiten especificar colores y fondos.

Colores

color: esta propiedad establece el color del texto. Puedes usar nombres de colores, códigos hexadecimales o valores RGB para definir el color del texto. 

                    
                        

                            color: red;

                            color: #00AABB;
                            
                            color: rgb(255, 0, 0);

background-color: esta propiedad establece el color de fondo de un elemento. 

                    
                        

                            background-color: #FFFFCC;

                            background-color: rgba(0, 128, 255, 0.5); /* con transparencia */

Fondos

background-image: esta propiedad te permite establecer una imagen como fondo en lugar de un color. Puedes proporcionar la ruta de la imagen como valor. 

                    
                        

                            background-image: url('imagen.jpg');

background-repeat: controla cómo se repite la imagen de fondo. Puedes utilizar valores como repeat, no-repeat, repeat-x, o repeat-y. 

                    
                        

                            background-repeat: repeat-x; /* Repetir solo en la dirección horizontal */

background-size: esta propiedad ajusta el tamaño de la imagen de fondo. Puedes usar valores como auto, contain, cover o especificar valores en píxeles o porcentajes. 

                    
                        

                            background-size: cover; /* Ajustar para cubrir completamente el elemento */

background-position: esta propiedad establece la posición inicial de la imagen de fondo. Puedes especificar valores en píxeles, porcentajes o palabras clave. 

                    
                        

                            background-position: center top; /* Centrar verticalmente y alineado arriba */

background: puedes combinar varias propiedades de fondo en una sola declaración. 

                    
                        

                            background: #FFFFCC url('imagen.jpg') no-repeat center center;

Puedes aplicar estas propiedades a elementos HTML específicos para personalizar la apariencia de tu sitio web, ya sea cambiando los colores de texto o estableciendo imágenes de fondo en tus elementos.

Diseño de Página con Flexbox

El diseño de página con Flexbox en CSS es una técnica que te permite crear diseños flexibles y eficientes para tu sitio web. Flexbox, o "Flexible Box Layout," es un modelo de diseño que simplifica la distribución y alineación de elementos en un contenedor, lo que facilita la creación de diseños responsivos y centrados en la alineación.

Conceptos Básicos de Flexbox

Contenedor Flex (Flex Container): el elemento padre que contiene los elementos flexibles. Debes aplicar display: flex; o display: inline-flex; al contenedor flex.

Elementos Flex (Flex Items): los elementos hijos del contenedor flex que se organizan y alinean según las reglas de Flexbox.

Eje Principal (Main Axis): la dirección principal en la que los elementos flex se alinearán. Por defecto, es horizontal (izquierda a derecha).

Eje Cruzado (Cross Axis): la dirección perpendicular al eje principal. Por defecto, es vertical (arriba a abajo).

Propiedades de Flexbox

display: flex;: Se aplica al contenedor flex y convierte a sus hijos en elementos flexibles.

flex-direction: Define la dirección del eje principal y puede ser row, row-reverse, column, o column-reverse.

justify-content: Controla la alineación de los elementos a lo largo del eje principal y puede ser flex-start, flex-end, center, space-between, o space-around.

align-items: Alinea los elementos a lo largo del eje cruzado y puede ser flex-start, flex-end, center, baseline, o stretch.

align-self: Permite ajustar la alineación de un elemento flexible individualmente, anulando align-items.

flex: Define la relación de crecimiento de los elementos flexibles y controla cómo se distribuye el espacio adicional.

Ejemplos de uso de Flexbox

Alineación Horizontal
                        
                            


                            

                            .container {
                                display: flex;
                                justify-content: center;
                            }
                              
                            .item {
                                flex: 1;
                            }

En este ejemplo, los elementos dentro del contenedor se alinearán horizontalmente en el centro.

Alineación Vertical
                        
                            

                                .container {
                                    display: flex;
                                    flex-direction: column;
                                    align-items: center;
                                }
                                  
                                .item {
                                    flex: 1;
                                }

En este caso, los elementos se alinearán verticalmente en el centro del contenedor.

Diseño de Cuadrícula
                        
                            

                                .container {
                                    display: flex;
                                    flex-wrap: wrap;
                                }
                                  
                                .item {
                                    flex: 1;
                                }

Este ejemplo permite crear un diseño de cuadrícula donde los elementos se envuelven automáticamente cuando el espacio es insuficiente.

Diseño de Página con CSS Grid

CSS Grid Layout es un sistema de diseño bidimensional en CSS que te permite crear diseños de página complejos y flexibles. A diferencia de Flexbox, que es un modelo de diseño unidimensional (una fila o una columna), CSS Grid permite organizar elementos en filas y columnas, lo como si fuera una cuadrícula.

Conceptos Básicos de CSS Grid

Contenedor Grid (Grid Container): es el elemento principal que contiene los elementos que se organizarán en una cuadrícula. Para convertir un elemento en un contenedor de cuadrícula, aplica display: grid; al mismo.

Elementos de Grid (Grid Items): son los elementos hijos del contenedor de cuadrícula que se organizan en filas y columnas de la cuadrícula.

Filas y Columnas: en CSS Grid, puedes especificar el número de filas y columnas, así como sus tamaños, utilizando propiedades como grid-template-rows y grid-template-columns.

Celdas: cada celda de la cuadrícula es una ubicación específica donde un elemento de la cuadrícula puede ubicarse.

Propiedades de CSS Grid

display: grid;: Se aplica al contenedor de cuadrícula para activar el modelo de cuadrícula.

grid-template-rows y grid-template-columns: Estas propiedades definen el número y el tamaño de las filas y columnas de la cuadrícula. Puedes especificar tamaños fijos, porcentajes o valores automáticos.

grid-template-areas: Te permite asignar nombres a áreas en la cuadrícula y organizar elementos en función de esas áreas.

grid-gap: Define el espacio entre las filas y las columnas de la cuadrícula.

grid-row y grid-column: Estas propiedades permiten ubicar elementos en ubicaciones específicas de la cuadrícula. 

                
                    

                        .container {
                            display: grid;
                            grid-template-columns: 1fr 1fr 1fr; /* Tres columnas de igual tamaño */
                            grid-template-rows: 100px 200px; /* Dos filas con tamaños fijos */
                            grid-gap: 10px; /* Espacio entre filas y columnas */
                        }
                                  
                        .item {
                            background-color: #3498db;
                            padding: 10px;
                            text-align: center;
                            color: white;
                        }
                                  
                        .item1 {
                            grid-row: 1 / 3; /* Ocupa dos filas */
                            grid-column: 1 / 2; /* Ocupa una columna */
                        }
                                  
                        .item2 {
                            grid-row: 1 / 2; /* Ocupa una fila */
                            grid-column: 2 / 4; /* Ocupa dos columnas */
                        }
                                  
                        .item3 {
                            grid-row: 2 / 3; /* Ocupa una fila */
                            grid-column: 2 / 4; /* Ocupa dos columnas */
                        }

En este ejemplo, el contenedor .container tiene una cuadrícula de tres columnas y dos filas con espacio entre filas y columnas. Los elementos .item1, .item2, y .item3 se ubican en ubicaciones específicas dentro de la cuadrícula utilizando las propiedades grid-row y grid-column.

CSS Grid proporciona un control preciso sobre el diseño de la página, lo que lo convierte en una herramienta poderosa para crear diseños complejos y flexibles.

Transiciones y Animaciones CSS

Las transiciones y animaciones en CSS son técnicas que te permiten añadir efectos visuales y dinamismo a tu sitio web.

Transiciones CSS

Las transiciones permiten cambiar gradualmente los valores de las propiedades CSS de un estado a otro. Puedes animar propiedades como el color, tamaño, posición y más. 

                    
                        

                            .button {
                                background-color: #3498db;
                                transition: background-color 0.3s ease;
                              }
                              
                              .button:hover {
                                background-color: #e74c3c;
                            }

En este ejemplo, cuando el cursor se sitúa sobre el botón, el color de fondo cambia de azul (#3498db) a rojo (#e74c3c) con una transición suave de 0.3 segundos (definida con transition) usando la función de temporización "ease".

Animaciones CSS

Las animaciones en CSS te permiten crear secuencias de cambios de estilo a lo largo del tiempo. Puedes definir keyframes (fotogramas clave) para especificar cómo cambian los estilos en diferentes momentos de la animación. 

                    
                        

                            @keyframes bounce {
                                0%, 20%, 50%, 80%, 100% {
                                  transform: translateY(0);
                                }
                                40% {
                                  transform: translateY(-30px);
                                }
                                60% {
                                  transform: translateY(-15px);
                                }
                            }
                              
                            .ball {
                                width: 50px;
                                height: 50px;
                                background-color: #f39c12;
                                animation: bounce 2s infinite;
                            }

En este ejemplo, hemos creado una animación llamada "bounce" que mueve una bola hacia arriba y abajo. La animación se aplica al elemento con la clase .ball y se repite infinitamente (infinite) durante 2 segundos.

Diferencias clave

Transiciones

Son adecuadas para cambios sencillos y para proporcionar retroalimentación interactiva.

Se basan en el cambio de una propiedad de estado a otro con una transición suave.

Animaciones

Son más versátiles y adecuadas para crear efectos complejos y secuencias de cambios en el tiempo.

Puedes definir keyframes para especificar estilos en varios momentos.

Puedes combinar transiciones y animaciones para crear experiencias de usuario más atractivas en tu sitio web.

Multimedia en la Web

Puedes incorporar imágenes(vistas anteriormente), audio y video en tu página web utilizando HTML y CSS. Ejemplos para audios y videos:

Audio

Incorporar un archivo de audio con HTML: 

                    
                        

                            <!DOCTYPE html>
                            <html>
                                <head>
                                    <title>Mi Página Web con Audio</title>
                                </head>
    
                                <body>
                                    <audio controls>
                                        <source src="musica.mp3" type="audio/mpeg">
                                    </audio>
                                </body>
                            </html>

Estilizar el reproductor de audio con CSS (personalizar la apariencia): 

                    
                        

                            audio {
                                width: 300px;
                                background-color: #f4f4f4;
                                border: 1px solid #ddd;
                                padding: 10px;
                            }

Video

Incorporar un archivo de video con HTML: 

                    
                        

                            <!DOCTYPE html>
                            <html>
                                <head>
                                    <title>Mi Página Web con Audio</title>
                                </head>

                                <body>
                                    <video width="640" height="360" controls>
                                        <source src="video.mp4" type="video/mp4">
                                    </video>
                                </body>
                            </html>

Estilizar el reproductor de video con CSS (personalizar la apariencia): 

                    
                        

                            video {
                                width: 640px;
                                height: 360px;
                                border: 1px solid #ddd;
                                background-color: #f4f4f4;
                            }

Estos son ejemplos básicos de cómo incorporar audio y video en tu página web utilizando HTML y CSS. Puedes personalizar aún más la apariencia y funcionalidad utilizando CSS y JavaScript. Además, asegúrate de que los archivos de medios (imágenes, audio y video) estén ubicados en las rutas correctas y en formatos compatibles con los navegadores.
